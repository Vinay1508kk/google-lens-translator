# 🌐 Google Lens + Translator (Streamlit App)

A smart **text detection and translation** app built with **Streamlit**, **EasyOCR**, and **Google Translate API** — works just like **Google Lens**! 📱

---

## 🚀 Features

* 📷 Upload or capture an image
* 🔍 Detect and extract text automatically
* 🌎 Translate extracted text into multiple languages
* 💾 Download both extracted and translated text
* 🎨 Beautiful Streamlit interface

---

## 🧩 Tech Stack

* Python 3.8+
* Streamlit
* EasyOCR
* OpenCV
* Googletrans
* Pillow

---

## ⚙️ Installation & Run

1. Clone this repo:

   ```bash
   git clone https://github.com/<your-username>/google-lens-translator.git
   cd google-lens-translator
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the app:

   ```bash
   streamlit run google_lens_translator.py
   ```

4. Open in browser:
   👉 [http://localhost:8501](http://localhost:8501)

---

## 🌍 Supported Languages

English, Hindi, Tamil, Telugu, French, Spanish (you can add more easily).
